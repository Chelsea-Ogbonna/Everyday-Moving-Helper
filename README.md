## Problem

Existing robotic cargo assistants and smart luggage systems use fixed cargo compartments, limiting their ability to securely transport items of varying sizes and shapes.

## Solution

Everyday Moving Helper: An expandable powered cargo-assistance platform designed to transport everyday items such as groceries, luggage, backpacks, laundry baskets, and moving boxes.

## Experimental Validation

Terrain testing was conducted using:

- MPU6050 IMU
- Force sensor
- 4WD Mecanum Wheel Car Chassis Kit

Terrains tested:

- Smooth flooring
- Rough terrain
- Carpet
